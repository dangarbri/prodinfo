# prodinfo

Info is returned in JSON and printed to the console.

## Example

{

  "name": "Barbeque Sauce Carolina",

  "fiber": "0",
  "sodium": "280",
  "sugars": "9",
  "protein": "0",
  "calories": "45",
  "trans_fat": "0.0",
  "saturated_fat": "0.0",
  "cholesterol": "0",
